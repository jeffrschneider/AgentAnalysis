Methods Matrix

How Agentic Replica and Agentic Reengineering map to Agent Story formats—choosing the right combination for your needs.

Understanding the Matrix

The methods (Agentic Replica and Agentic Reengineering) describe how you research and analyze work. The artifacts (Agent Story Light and Full) describe how you document the resulting agent specifications.

There is no single "correct" pairing—the right combination depends on your project's complexity, timeline, and goals. This matrix helps you understand which combinations work best for different scenarios.

Method-Artifact Matrix

Agent Story: Light Streamlined documentation
Agent Story: Full Comprehensive specification
Agentic Replica Current state analysis
Excellent Fit

Quick documentation of existing workflows. Perfect for capturing the "as-is" state efficiently.

  • • Single-purpose agents
  • • Proof of concept
  • • Rapid prototyping
Situational

Detailed documentation of complex existing roles. Use when the current workflow itself is sophisticated.

  • • Complex existing processes
  • • Compliance requirements
  • • Knowledge preservation
Agentic Reengineering Future state vision
Good Fit

High-level vision documentation for stakeholder alignment and early-stage planning.

  • • Executive communication
  • • Initial feasibility review
  • • Scope definition
Excellent Fit

Comprehensive specification for implementation. The natural progression for production-ready agents.

  • • Multi-agent systems
  • • Engineering handoff
  • • Production deployment

Recommended Pathways

1

Quick Automation Assessment

Need to rapidly evaluate whether a role can be automated and get a working prototype.

Agentic Replica → Agent Story: Light

Timeline: 1-2 weeks | Best for: Proof of concept, feasibility studies

2

Full Transformation Project

Complete analysis and redesign of a job area, culminating in production-ready specifications.

Agentic Replica → Agentic Reengineering → Agent Story: Full

Timeline: 4-8 weeks | Best for: Strategic transformation, complex roles

3

Greenfield Agent Design

Designing new AI-native workflows without an existing human process to replicate.

Agentic Reengineering → Agent Story: Full

Timeline: 3-6 weeks | Best for: New capabilities, AI-native processes

4

Iterative Refinement

Start simple and progressively elaborate as the project matures and requirements become clear.

Agentic Replica → Agent Story: Light → Reengineering → Agent Story: Full

Timeline: Ongoing | Best for: Agile teams, evolving requirements

What Each Method Contributes to Artifacts

From Agentic Replica

→ Agent Story: Light

  • Core story (trigger, action, outcome)
  • Skills inventory from observed tools
  • Basic trigger patterns from inputs
  • Success criteria from outputs

→ Agent Story: Full

  • Detailed trigger specifications
  • Behavior model from transformations
  • Human interaction patterns
  • Baseline acceptance criteria

From Agentic Reengineering

→ Agent Story: Light

  • Optimized autonomy level
  • Refined human role definition
  • Escalation conditions
  • Guardrails from ethical review

→ Agent Story: Full

  • Complete reasoning strategies
  • Memory and learning design
  • Multi-agent collaboration patterns
  • Quality metrics and benchmarks

Quick Decision Guide

If you need to... Use this method With this artifact
Quickly assess automation potential Agentic Replica Light

Pitch a redesign to stakeholders Agentic Reengineering Light
Build production multi-agent system Agentic Reengineering Full
Start simple, elaborate later Replica → Reengineering Light → Full
Design AI-native process (no human baseline) Agentic Reengineering Full

agentanalysis.net

Agent Analysis provides comprehensive frameworks and methodologies for understanding human-AI collaboration as the fundamental unit of modern work. As AI agents evolve from passive tools into capable partners with their own skills, judgment, and initiative, organizations face unprecedented challenges in designing effective collaborative systems. Our artifacts and methods address the full lifecycle of human-agent partnerships: establishing trust through transparent capability boundaries, allocating responsibilities dynamically based on context and competence, creating feedback loops that improve both human and agent performance, and building systems that evolve gracefully alongside advancing AI capabilities. Whether you're documenting existing workflows, designing new human-agent teams, or planning the transition from traditional software to agentic systems, Agent Analysis provides the structured approaches you need.
